# Where's Waldo?

Front-end for the Where's Waldo game. Built using React and styled with CSS modules.

## Installation

```bash
# Clone the repository
git clone git@github.com:goran1010/waldo-front-end.git

# Navigate to project directory
cd waldo-front-end

# Install dependencies
npm install

# Rename the environment file
cp .env.example .env
# Edit .env file as needed
```

## Usage

```bash
# Development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```
